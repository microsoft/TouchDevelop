Supported Browsers

The TouchDevelop Web App has been designed for mobile devices with touchscreens. It can also be used with keyboard and mouse.

TouchDevelop works in the most recent versions of various browsers:

  • Internet Explorer 10+
  • Chrome 22+ for PCs, Macs, Linux
  • Firefox 16+ for PCs, Macs, Linux
  • Safari 6+ for Macs
  • Mobile Safari on iOS 6+ for iPad, iPhone, iPod Touch
  • Chrome 19+ for Android
  • Chrome 38+ for iOS
  • Firefox 29+ for Android
  • Opera 18+ for Android (not Opera Mini)
  • Silk for Kindle Fire (2nd Generation) and up.
